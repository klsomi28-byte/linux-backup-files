# Backup Script (Linux)

## Description
A simple Bash script that creates a timestamped backup of files from a source directory to a destination directory.

## How It Works
- Takes files from the source folder
- Creates a new backup folder with date and time
- Copies all files into the backup folder

## Script Paths
- Source: `/mnt/WindowsDocuments`
- Destination: `/home/sanjana/WindowsBackup`

## How to Run
```bash
chmod +x backup.sh
./backup.sh
Output
Backup folder created with timestamp

Success message displayed after completion
